Kongu Engineering College Bus System

The KEC Bus Facility provides a modern RFID-based bus attendance system for Kongu Engineering College students. This system ensures safety, punctuality, and accurate attendance tracking for all students using college transport.

With smart automation, the KEC bus system helps prevent misuse, improves monitoring, and provides reliable transport services.
